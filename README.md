# Serve & Co website

Projet Fondement Multimédia  
2ème année cycle préparatoire intégré MPI

## Rapport de projet

Le rapport complet du projet est disponible dans le fichier `Serve_Co.pdf` :

- Ouvrez `Serve_Co.pdf` avec un lecteur de PDF (par exemple Adobe Acrobat Reader, votre navigateur web, ou tout autre lecteur PDF installé).
- Vous y trouverez le **rapport de projet** détaillant le contexte, les besoins, les choix techniques et la réalisation du site.

## Demo du site

Pour voir une démonstration du site, ouvrez la vidéo `Demo_Site_Web.MP4` :

- Sous Windows : double-cliquez sur `Demo_Site_Web.MP4` ou clic droit → *Ouvrir avec* → choisissez votre lecteur (par exemple VLC, Films et TV).
- Sous macOS : double-cliquez sur `Demo_Site_Web.MP4` ou clic droit → *Ouvrir avec* → choisissez QuickTime ou VLC.
- Sous Linux : double-cliquez sur `Demo_Site_Web.MP4` ou ouvrez-le directement avec votre lecteur vidéo (par exemple VLC, mpv).
